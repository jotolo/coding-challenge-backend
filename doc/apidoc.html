• BadiCodingChallengeBackend 1.0

API documentation for Badi's Coding Challenge Backend.

Resources

Armors

Resource Description
GET /api/armors Return all armors
GET /api/zombies/:zombie_id/armors Return all armors belonging to an specific zombie.
GET /api/armors/1 Return an specific armors
GET /api/zombies/:zombie_id/armors/1 Return an specific armors belonging to an specific zombie.
POST /api/armors Create an armor
POST /api/zombies/:zombie_id/armors Create an armor belonging to an specific zombie.
PUT /api/armors/1 Update an armor
PUT /api/zombies/:zombie_id/armors/1 Update an armor belonging to an specific zombie.
DELETE /api/armors/1 Destroy an armor
DELETE /api/zombies/:zombie_id/armors/1 Destroy an armor belonging to an specific zombie.

Weapons

Resource Description
GET /api/weapons Return all weapons
GET /api/zombies/:zombie_id/weapons Return all weapons belonging to an specific zombie.
GET /api/weapons/1 Return a weapon
GET /api/zombies/:zombie_id/weapons/1 Return a weapon belonging to an specific zombie.
POST /api/weapons Create an weapon
POST /api/zombies/:zombie_id/weapons Create an weapon belonging to an specific zombie.
PUT /api/weapons/1 Update an weapon
PUT /api/zombies/:zombie_id/weapons/1 Update an weapon belonging to an specific zombie.
DELETE /api/weapons/1 Destroy a weapon
DELETE /api/zombies/:zombie_id/weapons/1 Destroy an weapon belonging to an specific zombie.

Zombies

Resource Description
GET /api/zombies Return all zombies or search for specific zombies.
GET /api/zombies/1 Return an specific zombie
POST /api/armors Create an armor
PUT /api/zombies/1 Update a zombie
DELETE /api/zombies/1 Destroy a zombie
POST /api/zombies/1/add_armors Add armors to zombie
POST /api/zombies/1/add_weapons Add weapons to zombie
POST /api/zombies/1/eat_brain Allow zombies to eat brains.
