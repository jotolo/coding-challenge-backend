Zombies / index

GET /api/zombies
Return all zombies or search for specific zombies.

Return or search for zombies.

Params

  • page : optional [

    Must be a Integer

    ]

    Page number you want to list. Default: 1

  • per_page : optional [

    Must be a Integer

    ]

    Elements per page you want to list. Default: 30

  • name : optional [

    Must be a Integer

    ]

    Search by name.

  • hit_points : optional [

    Must be a Integer

    ]

    Search by hit_points.

  • speed : optional [

    Must be a Integer

    ]

    Search by speed.

  • brains_eaten : optional [

    Must be a Integer

    ]

    Search by brains_eaten.

  • turn_date : optional [

    Must be a Date

    ]

    Search by turn_date.

  • armor_ids : optional [

    Must be an array of any type

    ]

    Search zombies who has armors by IDs.

  • weapon_ids : optional [

    Must be an array of any type

    ]

    Search zombies who has weapons by IDs.
